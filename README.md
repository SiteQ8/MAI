**MAI (ماي)** — "water" in Kuwaiti dialect — is a mobile-first water delivery marketplace connecting Kuwait's residents with local water companies. Browse companies, compare products, order gallons, filters, dispensers, and subscribe to weekly/monthly delivery plans.

## ✨ Features

### 🏠 Home Screen
- **Personalized greeting** — Good morning/evening with user name
- **Smart search** — Search companies by Arabic or English name
- **Points dashboard** — Points earned + money saved at a glance
- **Category carousel** — 6 categories: Bottled, Gallons, Filters, Dispensers, Subscriptions, Accessories
- **Featured offers** — Horizontal scroll cards with discount percentages
- **Company directory** — All companies with ratings, delivery time, product count
- **Water quality tips** — Daily hydration tips and water facts
- **Delivery coverage map** — Areas served in Kuwait

### 🏢 Company Pages
- **Company hero** — Logo, name, rating, reviews, delivery time, minimum order
- **About section** — Company description and certifications
- **Category filter tabs** — Filter products by type
- **Product cards** — Name, price, old price (if on sale), badge (Offer/New/Best Seller)
- **Quick add to cart** — One-tap add with quantity controls
- **Working hours** — When the company delivers
- **Contact info** — Phone, WhatsApp, location

### 🛒 Cart & Checkout
- **Cart management** — Add, remove, update quantities
- **Promo codes** — Enter `MAI10` for 10% discount, `FIRST` for 15% first order
- **Order summary** — Itemized list with subtotal, delivery fee, discount
- **Multiple payment methods:**
  - 🏧 **KNET** — Kuwait's national payment network
  - 💳 **Credit/Debit Card** — Visa, Mastercard
  -  **Apple Pay** — One-tap payment
  - 💵 **Cash on Delivery** — Pay the driver
- **Delivery address** — Saved addresses with area/block/street
- **Delivery time slot** — Choose morning, afternoon, or evening
- **Order notes** — Special instructions for the driver
- **Points earned preview** — See how many points you'll earn

### 📦 Order Tracking
- **Active orders** — Live progress bar (Confirmed → Preparing → On the way → Delivered)
- **Past orders** — Full history with reorder button
- **Order details** — Items, prices, payment method, delivery time
- **Driver info** — Name and phone (demo)
- **Rate & review** — Star rating after delivery

### 👤 User Profile
- **Membership level** — Bronze → Silver → Gold with progress bar
- **Points system:**
  - Earn **10 points per 1 KD** spent
  - **Bronze** (0-999 points): Standard benefits
  - **Silver** (1000-4999 points): 5% bonus points, priority delivery
  - **Gold** (5000+ points): 10% bonus points, free delivery, exclusive offers
- **Stats dashboard** — Total orders, money saved, favorite company
- **Saved addresses** — Home, office, custom locations
- **Notification preferences** — Order updates, offers, new products
- **Referral program** — Share code, earn 500 points per referral
- **Order history** — All past orders with totals

### 🌐 Bilingual (AR/EN)
- Full Arabic and English support
- RTL/LTR layout switching
- All text, buttons, labels, and data in both languages
- One-tap language toggle

### 💡 Additional Features
- **Water quality facts** — Daily tips about hydration and water quality
- **Subscription plans** — Weekly/monthly gallon delivery
- **Favorites** — Save favorite products for quick reorder
- **Delivery areas** — Coverage map for all of Kuwait
- **Company comparison** — Compare prices across companies
- **Dark mode ready** — CSS variables for easy theming

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
git clone https://github.com/SiteQ8/MAI.git
cd MAI
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

### Demo Promo Codes

| Code | Discount | Conditions |
|------|----------|------------|
| `MAI10` | 10% off | Any order |
| `FIRST` | 15% off | First order only |
| `WATER50` | 50% off delivery | Orders over 5 KD |
| `SUMMER` | Free gallon | With 5+ gallon order |
